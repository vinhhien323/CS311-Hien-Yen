## II. How to Run the Project

1. **Data Preparation**:
   - Store the data for the retrieval step in the `data` folder.
   - Place PDF documents in the `data_PDF` folder.

2. **Install Required Libraries**:
   - Install all necessary libraries listed in the `requirements.txt` file using:
     ```bash
     pip install -r requirements.txt
     ```

3. **Configure API Key**:
   - Obtain a Google Gemini API Key.
   - Replace all strings initialized with `PLEASE ADD GEMINI API KEY HERE` in the code with your API key.

4. **Run the Application**:
   - Open your terminal and run the following command:
     ```bash
     streamlit run main_faiss.py
     ```
